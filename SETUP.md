## Obsidian
Obsidian is a powerful knowledge base that works on top of a local folder of plain text Markdown files.

1. Install Obsidian from [here](https://obsidian.md/download).
2. Install Node.js (>=v22) from [here](https://nodejs.org/en/download/) for Obsidian GitHubCopilot plugin.
3. Setting for plugin "Obsidian GitHubCopilot" to input "Path to Node binary". (example: C:\Program Files\nodejs\node.exe)
